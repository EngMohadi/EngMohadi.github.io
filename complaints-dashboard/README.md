# Municipal Complaints Smart Dashboard — داشبورد شكاوى المواطنين

داشبورد تفاعلي لشكاوى السكان خاص بالبلدية (منطقة الدراسة: Kolding – Denmark) مبني بـ **OpenLayers 10** و **Chart.js 4** بدون أي إطار عمل.

## التشغيل

يجب تشغيل الصفحة عبر خادم محلي (وليس فتح الملف مباشرة) بسبب تحميل ملفات GeoJSON:

```
cd Citizen_Complaints_Dashboard
python -m http.server 8123
```

ثم افتح: `http://localhost:8123`

## بنية المشروع

```
├── index.html            الصفحة الرئيسية (RTL عربي)
├── css/style.css         التنسيقات + تنسيق الطباعة (تقرير PDF)
├── js/app.js             منطق الداشبورد (خريطة، رسوم، فلاتر، تصدير)
├── data/
│   ├── complaints.geojson      886 شكوى (2025-01 → 2026-07)
│   ├── neighborhoods.geojson   10 أحياء (مولّدة بـ Voronoi حول تجمعات المباني)
│   ├── municipality.geojson    حدود منطقة الدراسة (طبقة cut من الـ GPKG)
│   ├── roads.geojson           637 شارعًا مسمّى
│   └── buildings.geojson       3467 مبنى
└── tools/generate_data.py      سكربت توليد البيانات من Denmark_sample.gpkg
```

## المزايا

- **الخريطة (OpenLayers):** نقاط الشكاوى ملوّنة حسب الحالة أو النوع، تلوين الأحياء حسب الكثافة (Choropleth) مع أسماء الأحياء وأعدادها، طبقات شوارع ومباني قابلة للتفعيل، نافذة تفاصيل عند النقر على أي شكوى.
- **المؤشرات (KPI):** إجمالي / مفتوحة / مغلقة / متأخرة / متوسط مدة الإغلاق / أكثر حي / أكثر نوع — والبطاقات قابلة للنقر كفلاتر سريعة.
- **الرسوم (Chart.js):** حسب النوع، حسب الحالة، الاتجاه الشهري 2025–2026 (مستلمة/مغلقة)، حسب الحي، أداء الأقسام (عدد + متوسط أيام الإغلاق)، مصدر الشكوى.
- **الجداول:** أكثر 10 شوارع شكاوى، وآخر الشكاوى (النقر على صف يقفز لموقعها على الخريطة).
- **الفلاتر:** التاريخ من/إلى، الحي، النوع، الحالة (شاملة "متأخرة فقط")، الأولوية.
- **التصدير:** Excel (ورقة الشكاوى + ورقة ملخص، RTL) وتقرير PDF (عبر الطباعة).

## منطق الشكاوى المتأخرة

`target_days` حسب الأولوية: عاجلة = 1، عالية = 3، متوسطة = 7، منخفضة = 14 يومًا.
الشكوى متأخرة إذا كانت غير مغلقة و `received_date + target_days < اليوم`.

## إعادة توليد البيانات

```
pip install shapely
python tools/generate_data.py
```

السكربت يقرأ الحدود والمباني والشوارع من `D:\DASH_BOARD\DataSamples\Denmark_sample.gpkg`،
يبني 10 أحياء (k-means + Voronoi)، ويولّد الشكاوى بأنماط موسمية واقعية
(مياه صيفًا، إنارة وطرق شتاءً، صحة وبيئة في الصيف...) ويصدّر GeoJSON إلى `data/`.
